
# Commentary and Interpretive Materials

This folder contains essays, tutorials, and public-facing commentary that interpret, contextualize, or translate the ReadingMachine method
for different audiences (researchers, practitioners, policy analysts, social scientists).

These materials are not methodological specifications and should not be read as claims about performance or validation.

The authoritative description of the system’s design, assumptions, and trade-offs is provided in the white paper located in `/documentation/`. The concise examples of how to run the pipelines are in `/examples/`
